# Contracts

Stable interfaces in this repository:

- `AxQxOS/TaskReceipt/v1`
- `AxQxOS/PipelineEvent/v1`
- `AxQxOS/WebhookEnvelope/v1`
- `AxQxOS/AntigravitySession/v1`
- `AxQxOS/LoRAReceipt/v1`
- `AxQxOS/YieldCurveReceipt/v1`

Schema files are in `schemas/contracts/*.schema.json`.
Use `scripts/validate_schemas.py` to validate payloads.
